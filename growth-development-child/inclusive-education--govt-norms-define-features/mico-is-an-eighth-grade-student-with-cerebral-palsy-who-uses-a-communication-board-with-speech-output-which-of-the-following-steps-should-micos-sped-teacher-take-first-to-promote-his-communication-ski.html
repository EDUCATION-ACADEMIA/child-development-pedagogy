MICO is an eighth-grade student with cerebral palsy who uses a communication board with speech output. Which of the following steps should Mico’s Sped teacher take FIRST to promote his communication skills in his new environment?

(A) Assigning students in a GEN ED class to write questions about MICO’s communication board and having him answer the questions

(B) Asking students in GEN ED class to discuss with the class different ways to communicate including the use of a communication board

(C) Having MICO introduces himself to the GEN ED class and explain when he starts using the communication board

(D) ** To ensure that the GEN ED teacher understands how MICO uses a communication board before he transitions to the class

EXPLANATIONS BELOW

Concept note-1: -Integrated education: It emphasizes the placement of children with disabilities in mainstream schools.

Concept note-2: -It provides a provision to include disabled children along with normal children in a regular classroom environment. It refers to an education system that accommodates all children regardless of their physical, intellectual, social, emotional, linguistic, or other conditions.
