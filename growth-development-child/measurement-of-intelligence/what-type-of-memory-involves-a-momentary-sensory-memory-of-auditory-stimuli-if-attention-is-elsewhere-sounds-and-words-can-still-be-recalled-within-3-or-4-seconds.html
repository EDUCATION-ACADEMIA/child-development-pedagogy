What type of memory involves a momentary sensory memory of auditory stimuli; if attention is elsewhere, sounds and words can still be recalled within 3 or 4 seconds?

(A) Iconic Memory

(B) ** Echoic Memory

(C) Explicit Memory

(D) Implicit Memory

EXPLANATIONS BELOW

Concept note-1: -Echoic Memory: a momentary sensory memory of auditory stimuli; if attention is elsewhere, sounds and words can still be recalled with 3 or 4 seconds. Short-term Memory: temporary memory storage.

Concept note-2: -Echoic memory is a momentary sensory memory of auditory stimuli where sounds and words can still be recalled within 3â€“4 seconds even if your attention is elsewhere.

Concept note-3: -Echoic memory, or auditory sensory memory, is a type of memory that stores audio information (sound).

Concept note-4: -Iconic Memory: A momentary sensory memory of visual stimuli; a photographic memory lasts no more than a few tenths of a second.

Concept note-5: -Iconic and echoic memory are two forms of sensory memory, which momentarily stores information from our senses before it is encoded in short-term memory. Iconic memory is the storage of what we see, while echoic memory is the storage of what we hear. Both of these functions can be improved with selective attention.
