If an object has momentum, you can be sure that it ____

(A) is heavy

(B) is going fast

(C) ** is moving

(D) none of these

EXPLANATIONS BELOW

Concept note-1: -To meet the definition of having momentum, an object must be moving. Momentum is mass in motion. To have momentum an object (which is guaranteed to have mass) must be moving.

Concept note-2: -It is the product of an objectâ€™s mass and velocity. An object at rest has a momentum of 0. This helps distinguish momentum from inertia. An object must be in MOTION to have momentum.

Concept note-3: -In order to have momentum, an object must be in motion. All objects that are moving have momentum, but objects that are not in motion have no momentum.

Concept note-4: -ANSWER: FALSE-An object has momentum if it is moving. Having mass gives an object inertia. When that inertia is in motion, the object has momentum. If an object does not have momentum, then it definitely does not have mechanical energy either.
