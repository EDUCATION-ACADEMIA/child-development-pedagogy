We can use memory card as a storage device in computer also in camera.

(A) ** True

(B) False

EXPLANATIONS BELOW

Concept note-1: -A Memory Card is a generic term for a card that you put into your camera or computer that has memory to hold digital media files like photos, music, or documents. An SD Card is a type of memory card popular with the majority of consumer camera manufacturers used to store the photos captured with the camera.

Concept note-2: -Memory cards are literally just miniature digital data storage devices. These cards house non-volatile flash storage within them, which simply means that they can retain stored information or data even after they’re unplugged-i.e., removed from a camera.

Concept note-3: -If you’re shooting HD-resolution video, your memory card should be at least a Class 10, U1, or V10-all of these have a minimum sustained speed of 10 MB/s. If you’re shooting 4K-resolution video, your memory card should be at least U3 or V30. Both of these have a minimum sustained speed of 30 MB/s.
