Whichof the following equations corresponds to the statement “x varies directly as the square of y” ?

(A) y = k x 2

(B) y = 2 kx

(C) ** x = k y 2

(D) x = k y 2

EXPLANATIONS BELOW

Concept note-1: -For direct variation, if x varies directly as y then x=my x = m y, where m is a proportionality constant. For inverse variation, if x varies inversely as y then x=ny x = n y, where n is a proportionality constant.

Concept note-2: -If y varies directly with x, then we can express their relationship to one another using the following formula: y = kx, where k is a constant. Therefore, if y varies directly as the square of x and the cube of z, we can write the following analagous equation: y = kx2z3, where k is a constant.

Concept note-3: -The phrase “ y varies directly as x” or “ y is directly proportional to x” means that as x gets bigger, so does y, and as x gets smaller, so does y.

Concept note-4: -For a population, the variance is calculated as ² = ( (x-)² ) / N. Another equivalent formula is ² = ( ( x²) / N )-². If we need to calculate variance by hand, this alternate formula is easier to work with.
