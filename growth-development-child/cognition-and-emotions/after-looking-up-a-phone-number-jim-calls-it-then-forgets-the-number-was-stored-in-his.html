After looking up a phone number, Jim calls it, then forgets. The number was stored in his

(A) ** Short-Term Memory

(B) Long Term Memory

(C) Visual Memory

(D) Echoic Memory

EXPLANATIONS BELOW

Concept note-1: -For the purpose of a discussion on memory loss, short term memory is equivalent to very recent memories, usually measured in minutes-to-days. Examples of short term memory include where you parked your car this morning, what you had for lunch yesterday, and remembering details from a book that you read a few days ago.

Concept note-2: -Short-term memory (or memory) refers to information processed in a short period of time. Long-term memory allows us to store information for long periods of time, including information that can be retrieved consciously (explicit memory) or unconsciously (implicit memory).

Concept note-3: -Working memory is short-term memory that’s used to execute specific tasks. Working memory is a system that’s designed to manipulate and use short-term memories. Working memory is attention that’s designed to manage short-term memory.

Concept note-4: -short-term memory, in psychology, the concept involving the extremely limited number of items that humans are capable of keeping in mind at one time. Of undeniable importance, the long-standing concept of “short-term memory” is one of the most researched topics in cognitive science.
