The relatively permanent and limitless storehouse of the memory system is called ____ memory.

(A) state-dependent

(B) ** long-term

(C) implicit

(D) flashbulb

EXPLANATIONS BELOW

Concept note-1: -Long-term Memory: the relatively permanent and limitless storehouse of the memory system.

Concept note-2: -Long-term memory refers to the transfer of information from short-term memory into long-term storage in order to create enduring memories. This type of memory is unlimited in capacity and stable-lasting for years or even a lifetime.

Concept note-3: -There are two types of long-term memory: declarative or explicit memory and non-declarative or implicit memory. Explicit memory refers to information that can be consciously evoked. There are two types of declarative memory: episodic memory and semantic memory.

Concept note-4: -Many think of long-term memory as a permanent “bank” within the brain. Once a memory arrives there, the mind stores it completely and indefinitely. In truth, this is not the case. Although the long-term memory process allows information to remain in the brain for an extended period, nothing in the brain avoids risk.

Concept note-5: -While long-term memory has a seemingly unlimited capacity that lasts years, short-term memory is relatively brief and limited. Short-term memory is limited in both capacity and duration.
