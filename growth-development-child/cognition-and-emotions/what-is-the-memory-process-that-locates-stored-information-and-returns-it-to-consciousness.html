What is the memory process that locates stored information and returns it to consciousness?

(A) ** retrieval

(B) encoding

(C) storage

(D) interference

EXPLANATIONS BELOW

Concept note-1: –Retrieval: the process of locating stored information and returning it to conscious thought.

Concept note-2: -Memory retrieval involves the interaction between external sensory or internally generated cues and stored memory traces (or engrams) in a process termed ‘ecphory’. While ecphory has been examined in human cognitive neuroscience research, its neurobiological foundation is less understood.

Concept note-3: -There are three main processes that characterize how memory works. These processes are encoding, storage, and retrieval (or recall).

Concept note-4: -Psychologists distinguish between three necessary stages in the learning and memory process: encoding, storage, and retrieval (Melton, 1963). Encoding is defined as the initial learning of information; storage refers to maintaining information over time; retrieval is the ability to access information when you need it.
