A weakness of the multi-store memory model is:

(A) ** It is far too simplistic, and doesn’t explain all of our memories

(B) It has no application to everyday life

(C) There is no evidence to support it

(D) It doesn’t explain how information gets into the long term memory

EXPLANATIONS BELOW

Concept note-1: -Weaknesses. The model is oversimplified, in particular when it suggests that both short-term and long-term memory each operate in a single, uniform fashion. We now know is this not the case. It has now become apparent that both short-term and long-term memory are more complicated that previously thought.

Concept note-2: -MSM can be compared to Working Memory. It is more simplistic than Working Memory, because it doesn’t split STM up into acoustic and visual systems.

Concept note-3: -According to the multi store model for memory, memory is divided into three sections, sensory, short term and long term. The function of sensory memory is retaining information long enough to decide whether or not it is worthy of further processing.

Concept note-4: -The Multi-store model describes the loss of memory through decay, which the Working Memory model doesn’t. Furthermore, the working memory model explains short-term memory in some detail while Multi-Store model has just laid out general steps of memory.
